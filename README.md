# TweetZone 🐦

A Twitter clone built with Python and Django.

## Features
- User Authentication (Login/Signup)
- Tweet Post, Edit, Delete
- Follow / Unfollow System
- User Profile Page

## Tech Stack
- Backend: Python, Django, FastAPI
- Database: MySQL
- Frontend: HTML, CSS, Bootstrap
